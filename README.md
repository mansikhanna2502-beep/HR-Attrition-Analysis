# HR Attrition Analysis

## 📊 Project Overview
Analyzed an HR dataset to identify key factors contributing to 
employee attrition using Python.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)

## 📁 Dataset
- HR Attrition dataset from GitHub
- 1,470 employee records

## 📌 Key Findings
- Overall attrition rate: ~16% (237 out of 1,470 employees)
- Male employees had higher attrition than female employees
- Sales Executives & Laboratory Technicians had highest attrition
- Employees who left had lower median monthly income
- Correlation heatmap revealed relationships between job level, 
  income and tenure
